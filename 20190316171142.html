api

经过一阵捣鼓，发现page部署有这两个接口,去到部署界面，上方的地址栏可取得

第一个用来部署
第二个用来检测是否部署完毕

部署

请求类型: POST
参数：

  • branch: 要的部署分支
  • build_directory: 要的部署目录，为空则为全部
  • force_https: 是否使用https

请求头:

  • Cookie
  • User-Agent
  • X-CSRF-Token
  • X-Requested-With

检测

请求类型: POST
参数：

  • branch: 要的部署分支
  • build_directory: 要的部署目录，为空则为全部
  • force_https: 是否使用https

请求头:

  • Cookie
  • User-Agent
  • X-CSRF-Token
  • X-Requested-With
